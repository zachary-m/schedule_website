Ramaz Schedule Help

If you are incountering problems:

  1. Make sure the date you entered is a date of a Sunday.
  2. Click the log out button in the top left corner and make sure the student ID you entered is correct.
  3. The student ID that you will enter is only found in the link in one of you schedules. It is not on your Ramaz photo ID card, meaning it probably will not start with the number 7.
  4. Make sure JavaScript and Cookies are enabled.
  5. Make sure you are only entering your student ID went using the special link.
  6. This website only works for students.
